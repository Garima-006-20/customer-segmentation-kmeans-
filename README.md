# Customer Segmentation Using K-Means Clustering

## Project Overview

This project implements Customer Segmentation using the K-Means Clustering algorithm. The objective is to group customers based on their purchasing behavior and characteristics, helping businesses understand customer patterns and develop targeted marketing strategies.

## Features

* Data preprocessing and feature scaling
* Customer segmentation using K-Means Clustering
* Five customer categories:

  * Average Customers
  * Premium Customers
  * Impulsive Customers
  * Careful Customers
  * Budget Customers
* Interactive Streamlit web application
* Real-time customer segment prediction

## Technologies Used

## How to Run

1. Install dependencies:

pip install -r requirements.txt

2. Run the Streamlit application:

streamlit run app.py

3. Open the browser and access:

http://localhost:8501

## Results

The K-Means model successfully segments customers into five distinct groups based on behavioral patterns. These segments can help organizations improve customer retention, targeted marketing, and business decision-making.
